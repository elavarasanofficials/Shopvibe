# 🛒 ShopVibe Ecosystem

A massive, futuristic, and dynamic e-commerce ecosystem built with Next.js, Turborepo, Tailwind CSS v4, and Framer Motion.

## 🚀 Ecosystem Architecture

This project is a **Monorepo** containing multiple applications and shared packages:

- `apps/web`: The Consumer-facing E-Commerce storefront (Liquid Glass Design).
- `apps/admin`: The SaaS Control Center for inventory and category management.
- `packages/ui`: Shared UI components, Design System tokens, and Tailwind configurations.

## 🛠️ Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Monorepo:** Turborepo
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Database:** Firebase Firestore
- **Testing:** Vitest & Playwright

## 📂 Documentation

All planning, architecture documents, and active task checklists are maintained in the `docs/` folder.
- `docs/implementation_plan.md`: The master architecture and screen flow.
- `docs/design_system.md`: UI/UX logic and mockups.
- `docs/task.md`: The active development checklist.
